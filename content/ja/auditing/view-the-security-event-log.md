# セキュリティイベントログを表示する

セキュリティログは、各オブジェクトに設定した監査ポリシーによって定義された各イベントを記録します。

**セキュリティログを表示するには**

1.  イベントビューアを開きます。
2.  コンソールツリーで **Windows ログ** を展開し、**セキュリティ** をクリックします。結果ペインには個々のセキュリティイベントが一覧表示されます。
3.  特定のイベントの詳細を確認したい場合は、結果ペインでそのイベントをクリックします。
